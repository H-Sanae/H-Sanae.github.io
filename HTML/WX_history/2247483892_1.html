什么是ASMR

Autonomous sensory meridian response

ASMR是Autonomous sensory meridian response 的缩写，可翻译为自发性感知神经反应，是一种神经科学方面的专业用语，指的是通过一些特定的外部音视频刺激获得的一种身体自发性愉悦反应，通常是从头皮处开始，随后传递到后脖颈再到脊椎上端的一种近似酥麻感的体验，这种体验感受是主观的，因此因人而异，有人对此很喜欢，而有的人却完全无感

那么具体都有哪些刺激可以让你获得ASMR触发体验呢？有人大概列举了以下几种：

triggers


耳边的低声细语，比如用很温柔的声音说一些鼓励的话，或一些特殊声音，如带有T、D、K等发音的词


抓挠声 或 敲击声，如手指轻轻敲击桌面或小物件时所发出来的声响


掏耳朵的声音 或 呼吸声，如耳搔的声音，触摸头发的声音，或在耳边轻轻吹气等与你有所接触时所发出的声音


木头燃烧的声音, 如木炭燃烧时发出的噼里啪啦声 或者 风声 海浪声 等其他自然之声


翻书时的翻页声、写字绘画时纸笔的摩擦声，比如缓慢的翻书配合嗓音轻柔舒缓的阅读声


角色扮演类，比如理发店或诊所，由于是角色扮演，所以会模拟真实情况下的声响，比如 剪头发的声音 或 医生问诊时的道具生或接触声

如今，ASMR已经帮助许多人在晚上放松并入睡，它的触发可以给头皮或身体带来愉悦的放松体验，所以现在很多喜欢ASMR的人都离不开它，只要一上床或一想休息就会戴上耳机，这很容易变成一种习惯，因为它是真的有效和舒适，所以你的大脑会经常提醒你，让你开启ASMR来帮助消除压力~

关于同人音声

同人音声是以声音为主的同人创作活动，这里创作的“声音”并不是以“音乐”为主，而是由音效、环境音、人声等组成的，多数带有情景剧情的声音作品(类似于广播剧)

RJ号是什么

DLsite Web ID


相信初次接触日语同人音声的小伙伴都会疑惑一个东西，你们说的RJ号是什么？我该如何获取作品的RJ号呢？那么本篇就将这个疑惑一并解答（如果有小伙伴向你提出类似的问题或者你想向其他小伙伴安利科普，那就把这篇狠狠砸过去吧）

其实RJ号就是DLsite同人作品页面的网页ID，形似于RJ337173这样的一串号码，与B站的BV号、P站的pid一样，都是指一个单独的网页ID，用来访问特定的网页内容


作为同人音声作品最大舞台的DLsite，其RJ号因为具有唯一性和准确性，并且简单识别不烧脑，因此现在很多人都直接用RJ号来指代某一部具体的同人音声作品


RJ号也可用于判断作品发售的相对时间，通常来说，越早期的作品RJ后面接的数字就越小，同理，数字越大说明该作品发售日期距离现在越近，也就是越新（但并不能代表质量越好！！）


有一点要知道的就是，同人音声的贩售网站并不只有DLsite一个，所以其他网站的同人作品如果DL站没有贩售的话，也就不存在RJ号了。同样的，油管、斗鱼、虎牙、B站等平台上也有人在自制ASMR同人视频，这些也都是没有RJ号的哦~
